# Lightweight Landing Page

This project is a **simple and lightweight landing page** built using **HTML, CSS, and vanilla JavaScript**.  
It is designed to be fast, responsive, and easy to maintain, making it ideal for product presentations, personal portfolios, startups, or promotional websites.

---

## Features

- Clean and modern design
- Fully responsive layout
- Lightweight and fast loading
- No frameworks or heavy libraries
- Smooth scrolling and basic interactivity
- Cross-browser compatibility
- Easy to customize and deploy

---

## Technologies Used

- **HTML5** – Structure and semantic content  
- **CSS3** – Styling, layout, and responsiveness  
- **JavaScript** – Basic interactivity and UI behavior
